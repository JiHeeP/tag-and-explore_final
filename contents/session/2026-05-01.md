## 남은 주의사항

| 항목 | 상태 | 설명 |
|------|------|------|
| Supabase 무료 플랜 Cached Egress | 주의 | Supabase Storage 대신 Cloudflare R2로 이미지 저장을 옮겨 부담을 낮춤 |
| Confirm email 해제 | 개발용 | 실제 서비스 공개 전에는 SMTP 설정 또는 이메일 확인 정책을 다시 검토해야 함 |
| 테스트 데이터 | 주의 | codex-hotspot-flow-test 등 테스트용 콘텐츠가 남아 있을 수 있음 |
| 삭제 기능 | 안전장치 있음 | 앱 안에서 "삭제" 후 "정말 삭제"를 눌러야 최종 삭제됨 |
| 권한 정책 | 유지 필요 | RLS를 끄거나 owner_id 정책을 바꾸면 타인 작업 수정 위험이 생김 |

## 제시한 대안

| 대안 | 판단 | 이유 |
|------|------|------|
| Supabase Storage 계속 사용 | 보류 | Cached Egress 경고가 있어 이미지 저장소로는 부담 가능성 있음 |
| Cloudflare R2 사용 | 채택 | 이미지 파일 저장/전송 비용 부담을 줄이기 좋음 |
| Firebase 사용 | 보류 | 구글 로그인에는 강점이 있지만 현재 DB/RLS 구조를 다시 설계해야 함 |
| 이메일 확인 유지 | 보류 | Supabase 메일 rate limit 때문에 개발 테스트가 자주 막힘 |
| 이메일 확인 해제 | 채택 | 개발 단계에서 가입/로그인 검증이 쉬움 |
